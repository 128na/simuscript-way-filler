# 説明

指定範囲を選択した軌道で埋めるスクリプトです。
線路以外にも道路や滑走路、運河など全タイプで使用可能です。

# 使い方

この readme.txt のあるフォルダをpak(pak128など)/tool に入れる。
cursor.way_filler.pak をpak(pak128など)に移動する。
Simutransを起動してスロープツールにアイコンが表示されていれば導入成功です。

- ツールーバーのway_fillerをクリックしてスクリプトを起動
- マップ内の敷設したい軌道をクリック
- 敷設したい範囲をドラッグ
- 選択範囲のうち長い辺に向かって軌道が敷設されます

# 既知の問題

- アイコンがスロープツールにしか表示できなかったのでスロープツールに配置されています。
  → 特殊建築物ツールに移したいですね！
- 敷設するたびに軌道を選択する必要があります。
  → 連続して敷設させたいですね！
