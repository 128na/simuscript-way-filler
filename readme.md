# スクリプト置き場

## 設置定義方式

マップ内の特定座標に軌道などを設置することで敷設対象を選択させるタイプのスクリプト。

アドオンやpakセットに依存しないため、高度の柔軟性を維持しつつ臨機応変に利用できる。
